COMPARATIVE POLITICAL DATA SET 2014

This graph shows 36 democratic OECD countries by political leaning and the percentage of debt to GDP Ratio.

Each node is a country. The colors represent where each country falls on the political spectrum.

Source: Armingeon, Klaus, Christian Isler, Laura Knöpfel, David Weisstanner and Sarah Engler. 2016. Comparative Political Data Set 1960-2014. Bern: Institute of Political Science, University of Berne.
